## About

**Guessing Game** is a multiplayer browser game where players try to guess a YouTube video using only a short audio clip.

Players can create or join a lobby, choose a YouTube channel or a specific video, and compete over several rounds. During each round, the game plays a randomly selected audio fragment while hiding the video's title and image. Players submit their guesses, earn points based on accuracy, and the winner is determined after all rounds are completed.

### Features

- 🎵 Guess YouTube videos by audio only
- 👥 Multiplayer lobbies with room codes
- 🎮 2–4 player support
- ⚙️ Configurable clip duration
- 🔀 Random audio fragments
- 🏆 Scoreboard and winner screen
- 📱 Modern responsive interface
